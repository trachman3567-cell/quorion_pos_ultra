QuorionPOS Ultra - skeleton project (C2)
---------------------------------------
This project is a full POS skeleton intended to mimic Quorion-style UI and features.

How to use:
1. Extract ZIP
2. Open project folder in Android Studio (File -> Open)
3. Sync Gradle (internet required to download plugins)
4. Implement adapters and UI logic (TODO markers present)
5. Build APK: Build -> Generate APK(s)

Notes:
- Project includes Room entities and DAOs, Raw TCP printer helper.
- This is a template; you'll need to implement product adapter, cart logic, receipt formatting.
